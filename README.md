dptutils
========
 
A set of utility methods.
 
This library contains utility methods for:
 * Checksumming (calculating and diffing)
 * JNI library loading (dll/so files from jar)
 * Files
 * Resources
 * Sockets
 * Streams
 * XML
 
It is a work in progress.
